Installation Guide

  1. Upload the Theme: Compress face-eternity-theme/ into a ZIP file. Go to Appearance > Themes > Add New > Upload Theme in WordPress admin. Activate after installation.
  2. Upload Media: Copy the contents of uploads/face-eternity/ to wp-content/uploads/face-eternity/ on your server, or use Media > Add New.
  3. Import Content: Go to Tools > Import > WordPress. Upload face-eternity-export.xml. Check "Download and import file attachments." Click Submit.
  4. Set Homepage: Go to Settings > Reading. Set "Your homepage displays" to "A static page." Select "Home" as homepage and "Blog" as posts page.
  5. Configure Menus: Go to Appearance > Menus. Assign the imported Primary Menu to the "Primary Menu" location.
  6. Customize: Go to Appearance > Customize to set up the logo, hero text, social links, and Google Analytics ID.
  7. Set Permalinks: Go to Settings > Permalinks and select "Post name" for clean URLs.
